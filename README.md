# Island Data Bottle Trading Bot

这是一个用于 Island Data Bottle 交易比赛的自动交易机器人。该机器人能够处理多种产品的交易，包括：

- SQUID_INK（鱿鱼墨水）
- RAINFOREST_RESIN（雨林树脂）
- KELP_SEED（海藻种子）

## 功能特点

- 自动市场做市
- 动态定价策略
- 风险管理
- 仓位控制

## 项目结构

- `Trader.py`: 主要的交易逻辑实现
- `data_analysis.ipynb`: 数据分析和回测笔记本

## 使用方法

1. 克隆仓库
2. 安装依赖
3. 运行交易机器人

## 依赖项

- Python 3.x
- NumPy
- Jupyter Notebook（用于数据分析）
